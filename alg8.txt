Algorithm for Temperature Conversion:
- Start
- Read choice (c for Celsius to Fahrenheit, f for Fahrenheit to Celsius)
- If choice == 'c':
• Read Celsius value
• Fahrenheit = (Celsius * 9/5) + 32
• Display Fahrenheit
- Else if choice == 'f':
• Read Fahrenheit value
• Celsius = (Fahrenheit - 32) * 5/9
• Display Celsius
- Else: Display "Invalid choice"
- End
